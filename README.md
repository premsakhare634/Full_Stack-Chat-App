# 💬 Full Stack Real-Time Chat Application

A modern, real-time chat application built using the MERN stack with Socket.IO for instant messaging. This project demonstrates full-stack development skills including authentication, real-time communication, and media handling.

---

## 🚀 Features

* 🔐 JWT Authentication (Secure login & signup)
* 💬 Real-time messaging using Socket.IO
* 🟢 Online/Offline user status
* 📩 One-to-one chat functionality
* 🖼️ Image sharing via Cloudinary
* 🍪 Secure cookies for session management
* ⚡ Zustand for global state management
* 🎨 Clean and responsive UI

---

## 🛠️ Tech Stack

### Frontend

* React.js
* Zustand (State Management)
* Tailwind CSS / CSS
* Axios

### Backend

* Node.js
* Express.js
* MongoDB (Mongoose)

### Real-Time & Media

* Socket.IO
* Cloudinary

### Authentication

* JWT (JSON Web Tokens)
* Cookies

---

## 📂 Project Structure

/client → React frontend
/server → Node.js backend
/models → Mongoose schemas
/routes → API routes
/controllers → Business logic
/middleware → Auth & error handling

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```bash
git clone https://github.com/premsakhare634/Full_Stack-Chat-App.git
cd Full_Stack-Chat-App
```

### 2️⃣ Setup Backend

```bash
cd backend
npm install
```

Create a `.env` file:

```env
PORT=5000
MONGO_URI=your_mongodb_uri
JWT_SECRET=your_secret_key
CLOUDINARY_CLOUD_NAME=your_name
CLOUDINARY_API_KEY=your_key
CLOUDINARY_API_SECRET=your_secret
```

Run backend:

```bash
npm run dev
```

---

### 3️⃣ Setup Frontend

```bash
cd frontend
npm install
npm run dev
```
